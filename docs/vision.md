# Vision

## Project name

Mark-me-down

## Purpose

Turn messy notes into useful markdown.

People frequently write quick notes, save them somewhere, and later struggle to understand or organize them.

Mark-me-down solves this problem by transforming raw text into readable markdown while preserving the original meaning.

## Principles

* Single-purpose application.
* Human remains in control.
* Conservative rewriting.
* Markdown only.
* No chat interface.
* No conversation history.
* No memory.
* No autonomous behavior.
* Output must remain editable.

## User philosophy

Users are expected to:

1. Paste or import text.
2. Trigger one transformation.
3. Review the result.
4. Edit manually if desired.
5. Copy or download the markdown.

Further refinements are performed by re-running the process from scratch.
